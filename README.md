# Prompts - Canal Multiplataforma de Prompts para IA

Colección curada de prompts útiles y accionables para Claude, Gemini y ChatGPT.

## Estructura

```
data/
  prompts.json          # Base de datos de todos los prompts
  [categoría]/          # Prompts organizados por categoría
    [id].md             # Cada prompt en detalle
output/
  videos/               # Vídeos generados automáticamente
scripts/                # Scripts de automatización
web/                    # Sitio web (próximamente)
```

- Productividad
- Escritura creativa
- Programación
- Cocina
- Fitness
